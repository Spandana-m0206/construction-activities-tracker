# 🏗️ Construction Activities API

This project is a **backend API** for managing and tracking construction-related activities. It helps organize the workflow, monitor progress, and manage data related to workers, materials, tasks, and service areas.

## 🔧 Key Features

- ✅ Manage construction services like **plumbing, electrical work, carpentry**, etc.
- 📍 Maintain **service areas** (by city or region)
- 🧑‍🔧 Assign and track **worker specializations**
- 📅 Track and update daily **work activities**
- 🛠 Built using **Node.js**, **Express**, and **MongoDB**
- 🔐 Secured with **JWT authentication**
- 🧱 Scalable and modular API architecture

## 📁 Modules

- **Auth** – Register, login, and manage users
- **Services** – CRUD operations on service types (e.g., plumbing, painting)
- **Service Areas** – Define and manage geographical work zones
- **Specializations** – Manage skill-based worker classification
- **Activities** – Track progress and status updates of ongoing work
